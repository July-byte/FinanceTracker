# 📊 Finance Tracker (C# / .NET)

Консольное приложение для учёта личных доходов и расходов с возможностью аналитики и сохранения данных в формате JSON.

Проект выполнен в рамках практики по изучению C# и принципов объектно‑ориентированного программирования.

---

## 🎯 Цель проекта

Разработать консольное приложение для управления личными финансами с использованием:

- объектно‑ориентированного подхода
- коллекций и LINQ
- сериализации данных в JSON
- базовой аналитики и обработки данных

---

## ✅ Задачи проекта

- Реализовать добавление доходов и расходов
- Реализовать хранение операций в коллекции `List<T>`
- Добавить автоматическое сохранение и загрузку данных в JSON
- Реализовать сортировку операций (по дате и сумме)
- Добавить фильтрацию по категории
- Реализовать аналитический отчёт с агрегированием данных
- Организовать структуру проекта с разделением ответственности

---

## 🛠 Используемые технологии

- C#
- .NET 8
- ООП (классы, свойства, enum)
- LINQ
- Работа с файлами
- System.Text.Json
- Обработка исключений

---

## 📌 Реализованный функционал

- ➕ Добавление доходов и расходов  
- 📋 Просмотр всех операций  
- 🔍 Фильтрация по категории  
- ↕ Сортировка по дате и сумме  
- 📊 Аналитический отчёт:
  - общий доход
  - общий расход
  - баланс
  - крупнейшая трата
  - группировка расходов по категориям  
- 💾 Автоматическое сохранение данных в JSON  
- 📂 Загрузка данных при запуске программы
